## Installation

Pour exécuter ce projet localement :

1. Cloner le dépôt :
   ```bash
   git clone git@github.com:jonathanbensadoun/portfolio.git
   ```
2. Aller dans le répertoire du projet :
   ```bash
   cd portfolio
   ```
3. Installer les dépendances :
   ```bash
   npm install
   ```
4. Démarrer le serveur de développement :
   ```bash
   npm run dev
   ```
